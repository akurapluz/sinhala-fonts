# AkuraPluz — Sinhala Font CSS Library

> Use beautiful Sinhala fonts on any website — just like FontAwesome.

[![npm](https://img.shields.io/npm/v/akurapluz)](https://www.npmjs.com/package/akurapluz)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

## Install

```bash
npm install akurapluz
```

Or via CDN:

```html
<link rel="stylesheet" href="https://cdn.akurapluz.dulaksha.com/style.css">
```

## Usage

Add the class to any element:

```html
<p class="font-bindumathi">ආයුබෝවන් !</p>
```

## Available Fonts

| Font Family | CSS Class |
|---|---|
| bindumathi | `.font-bindumathi` |
| malithi | `.font-malithi` |
| Roboto-Bold | `.font-roboto-bold` |
| emanee | `.font-emanee` |

## WordPress Integration

Fonts are managed via the **AkuraPluz WordPress Theme** — upload fonts from the dashboard and they are automatically published to npm and served via CDN.
